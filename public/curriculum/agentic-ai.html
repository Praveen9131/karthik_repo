Course syllabus

Agentic AI

Designing, building and shipping autonomous multi-agent systems across every major framework, with tool interoperability via MCP and agent interoperability via the A2A protocol.

6 WEEKSLANGCHAIN · LANGGRAPH · CREWAI · AUTOGEN · ADK · MCP · A2A

Course content

9 sections • 55 lectures total
M-01 Agent Fundamentals 6 lectures • Week 1
Anatomy of an agent: LLM + tools + memory + planning loop
ReAct, plan-and-execute, reflection
Tool / function calling deep dive
Memory: short-term, long-term, episodic, vector
Human-in-the-loop patterns
Design patterns: router, orchestrator–worker, evaluator–optimizer
M-02 LangChain 6 lectures • Weeks 1–2
Chat models, prompts, output parsers
LCEL & Runnables; chains & memory
Document loaders, splitters, retrievers
Tools & toolkits; custom tool creation
Tool-calling agents
LangSmith tracing & debugging
M-03 LangGraph 8 lectures • Weeks 2–3
Why graphs beat linear chains
StateGraph: nodes, edges, conditional edges
State management, reducers, message state
Cycles, branching, parallelism
Checkpointing & persistence (cross-session memory)
Human-in-the-loop interrupts; subgraphs & multi-agent graphs
Streaming: values, updates, tokens
Builds: research agent, support agent, self-correcting RAG (CRAG / Self-RAG)
M-04 CrewAI 6 lectures • Weeks 3–4
Crews, Agents, Tasks; sequential vs hierarchical processes
Role-based design: role, goal, backstory
Built-in & custom tools
Flows (event-driven orchestration)
Memory & knowledge in CrewAI
Builds: content-creation crew, market-research crew
M-05 AutoGen 6 lectures • Week 4
ConversableAgent, AssistantAgent, UserProxyAgent
Two-agent chats, group chats, GroupChatManager
Sandboxed code-execution agents
AutoGen Studio
Conversation patterns: sequential & nested chats
Build: multi-agent coding assistant
M-06 Google ADK 6 lectures • Week 5
ADK architecture: agents, tools, runners, sessions
LlmAgent + workflow agents (Sequential, Parallel, Loop)
Tool ecosystem & Google service integrations
Multi-agent hierarchies
Deployment: Vertex AI Agent Engine
Agent evaluation in ADK
M-07 MCP — Model Context Protocol 6 lectures • Weeks 5–6
What MCP standardizes: agent ↔ tools/context
Architecture: hosts, clients, servers
Primitives: tools, resources, prompts
Building an MCP server (Python SDK / FastMCP)
Connecting servers to Claude Desktop, IDEs & agents
Transports: stdio vs HTTP/SSE; security considerations
M-08 A2A — Agent-to-Agent Protocol 6 lectures • Week 6
Why agent interoperability matters
Agent Cards, tasks, messages, artifacts
A2A vs MCP: complementary layers (MCP = agent↔tools, A2A = agent↔agent)
Discovery & capability negotiation
Building A2A-compliant agents
Interop demo: LangGraph agent ↔ CrewAI agent over A2A
M-09 Production Agentic Systems 5 lectures • Week 6
Agent evaluation: task success rate, trajectory evaluation
Observability & tracing for agents
Cost & latency control; guardrails
Safety: sandboxing, permissioning, approval workflows
Framework comparison: LangGraph vs CrewAI vs AutoGen vs ADK
