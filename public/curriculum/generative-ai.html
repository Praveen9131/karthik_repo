Course syllabus

Generative AI

Working with large language models in production: APIs, prompting, embeddings, retrieval-augmented generation, fine-tuning, and multimodal systems.

6 WEEKSLLMS · RAG · FINE-TUNING

Course content

5 sections • 33 lectures total
M-01 LLM Foundations 7 lectures • Week 1
How LLMs work: pretraining, tokenization (BPE), context windows
Instruction tuning, RLHF, DPO (concepts)
Model landscape: GPT, Claude, Gemini, Llama, Mistral; open vs closed
Sampling: temperature, top-p, top-k, penalties
Limitations: hallucination, cutoff, context limits
APIs: OpenAI, Anthropic, Google Gemini
Structured outputs, JSON mode, function / tool calling
M-02 Prompt Engineering 6 lectures • Week 2
Zero-shot, few-shot, chain-of-thought
System prompts, role prompting, output formatting
ReAct prompting; self-consistency
Prompt templates & versioning
Prompt injection & jailbreak awareness (security)
Evaluating prompt quality
M-03 Embeddings & RAG 8 lectures • Weeks 3–4
Embedding models: OpenAI, Cohere, sentence-transformers
Similarity: cosine, dot product, Euclidean
Vector DBs: ChromaDB, FAISS, Pinecone, Qdrant, Weaviate, pgvector
Chunking: fixed, recursive, semantic, document-aware
End-to-end RAG pipeline
Advanced RAG: hybrid search (BM25 + dense), reranking, query rewriting, HyDE, parent-document retrieval
Multimodal RAG (images + text)
Evaluation: RAGAS, faithfulness, answer relevance
M-04 Fine-Tuning & Open-Source LLMs 6 lectures • Week 5
Fine-tune vs RAG vs prompting — decision framework
PEFT: LoRA, QLoRA
Quantization: GGUF, GPTQ, AWQ, bitsandbytes
Local models: Ollama, vLLM, llama.cpp
Hugging Face: datasets, transformers, TRL
Instruction dataset preparation
M-05 Multimodal & Production GenAI 6 lectures • Week 6
Diffusion models; Stable Diffusion / DALL·E workflows
Vision-language models (GPT-4V, Claude vision, Gemini)
Speech: Whisper (STT), TTS
Observability: LangSmith, Langfuse
Caching, rate limiting, cost optimization
Guardrails & moderation; LLM-as-judge evaluation
Course project

Production-grade RAG chatbot over custom documents — hybrid search, reranking, citations, an evaluation suite, and a Streamlit / FastAPI frontend.
