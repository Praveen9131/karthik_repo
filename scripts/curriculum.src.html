Fundamentals of Programming

6 weeksNo prerequisites

Absolute-beginner-friendly course covering everything from your first program to full object-oriented design, exception handling, and clean-code principles.

M-01 Getting Started with Programming Week 1
  • How programs run: interpreter vs. compiler, bytecode & execution
  • Setup: editor / IDE, notebooks, Git & GitHub basics
  • Variables, data types, type casting, id() & mutability
  • Input/output, f-strings and formatting
  • Operators: arithmetic, relational, logical, bitwise, identity, membership
  • Number systems: binary, decimal, hexadecimal
  • Linux / terminal essentials
M-02 Control Flow Week 2
  • if / elif / else, nested conditions, ternary expressions
  • match-case (structural pattern matching)
  • for & while loops, range(), enumerate(), zip()
  • break, continue, loop-else
  • Pattern-printing problems (nested loop practice)
  • Modulo arithmetic & basic math for programming
M-03 Functions & Recursion Week 3
  • Defining functions, return values, docstrings
  • Default, keyword, *args and **kwargs
  • Scope: local, global, nonlocal; closures
  • Lambda, map, filter, reduce
  • Recursion: factorial, Fibonacci, Tower of Hanoi
  • Decorators & generators (yield) — intro
M-04 Built-in Data Structures Week 4
  • Strings: immutability, slicing, methods
  • Lists & tuples: operations, sorting with key functions
  • Dictionaries & sets: methods, comprehension patterns
  • List / dict / set comprehensions
  • collections module: Counter, defaultdict, deque, namedtuple
  • Shallow vs deep copy
M-05 Object-Oriented Programming Weeks 5–6

Core OOP

  • Classes, objects, __init__, self
  • Encapsulation: name mangling, @property getters/setters
  • Abstraction: ABC module, abstract methods
  • Inheritance: single, multiple, multilevel, MRO & super()
  • Polymorphism: overriding, duck typing, operator overloading
  • Instance vs class vs static methods; class attributes
  • Dunder methods: __str__, __repr__, __eq__, __len__, __iter__
  • Composition vs inheritance; dataclasses

Robust Programs

  • Exception handling: try/except/else/finally, custom exceptions
  • File handling: text, CSV, JSON; context managers (with)
  • Modules, packages & dependency / environment management
  • Type hints & mypy basics
  • Design patterns: Singleton, Factory, Observer, Strategy
  • SOLID principles overview
Course Project

Console-based Library Management System (or Bank Account System) built with full OOP — inheritance, polymorphism, custom exceptions, and JSON/CSV persistence.

Competitive Programming

6 weeksWeekly contests

Contest-oriented training on Codeforces, CodeChef and AtCoder — advanced algorithms, math, and speed, with practical tips for staying within tight time limits.

Weekly participation in at least one rated contest is mandatory, followed by upsolving of unsolved problems.
M-01 CP Essentials Week 1
  • Fast I/O: sys.stdin.readline, sys.stdout, input buffering
  • Runtime choices for contests; recursion limits & iterative rewrites
  • Key libraries: heapq, bisect, collections, itertools, math, functools
  • sortedcontainers (SortedList / SortedDict) as ordered-set replacement
  • Overflow-free big integers; modular arithmetic (mod 1e9+7)
  • Contest strategy: reading fast, edge cases, penalty math
M-02 Number Theory & Math Weeks 1–2
  • GCD/LCM, extended Euclidean algorithm
  • Sieve of Eratosthenes, segmented sieve, smallest prime factor
  • Prime factorization, divisor counting
  • Binary exponentiation; modular inverse (Fermat's little theorem)
  • Combinatorics: nCr with factorial precompute, inclusion–exclusion, pigeonhole
  • Euler's totient, Chinese Remainder Theorem
  • Matrix exponentiation (fast Fibonacci)
  • Probability & expected value problems
M-03 Bit Manipulation Week 2
  • Set / clear / toggle / check bits; popcount
  • XOR properties & classic XOR problems
  • Subset enumeration with bitmasks
  • Bitmask DP applications
M-04 Advanced Graphs & Trees Weeks 3–4
  • Binary lifting; LCA in O(log n)
  • Euler tour & tree flattening
  • Heavy-Light Decomposition (concept + basics)
  • Small-to-large merging; rerooting technique
  • Max-flow: Ford-Fulkerson, Edmonds-Karp, Dinic's (intro)
  • Bipartite matching; 2-SAT (intro)
M-05 Advanced Data Structures Week 4
  • Sparse tables (idempotent range queries)
  • Segment tree with lazy propagation — harder problems
  • Merge sort tree; persistent segment tree (concept)
  • Mo's algorithm (offline queries)
  • Sqrt decomposition
M-06 Advanced Techniques, Strings & Geometry Week 5
  • Meet in the middle; ternary search; randomization
  • Suffix arrays & LCP array
  • Aho–Corasick multi-pattern matching (intro)
  • Game theory: Nim, Grundy numbers, Sprague–Grundy
  • Geometry: cross product, convex hull, line intersection
M-07 Contest Bootcamp Week 6
  • 2–3 virtual contests per week + upsolving discipline
  • Editorial-reading strategy
  • Rating roadmap: Newbie → Pupil → Specialist → Expert
  • Translating CP skills into FAANG-style interviews
Course Goal

Reach a stable Codeforces Specialist rating (1400+) and solve 100+ contest problems across CF / CodeChef / AtCoder / CSES.

Data Science: ML to Deep Learning

14 weeksNumPy · Pandas · scikit-learn · PyTorch

From data wrangling and statistics through classical machine learning to deep learning and Transformers — every model implemented, evaluated, and deployed.

M-01 NumPy Week 1
  • ndarrays, dtypes, shape manipulation, broadcasting
  • Indexing, slicing, boolean masking, fancy indexing
  • Vectorized ops & universal functions
  • Linear algebra: dot, matmul, eigenvalues, SVD
  • Random module & reproducibility
M-02 Pandas Weeks 1–2
  • Series & DataFrames; reading CSV, Excel, JSON, SQL, Parquet
  • loc / iloc, filtering, query()
  • Missing data: fillna, dropna, interpolation
  • GroupBy, aggregation, pivot tables, crosstabs
  • Merge, join, concat
  • apply, map, vectorized string ops
  • Time series: datetime index, resampling, rolling windows
  • Memory optimization & categorical dtypes
M-03 Visualization Week 2
  • Matplotlib: figures, axes, subplots, customization
  • Seaborn: distributions, categorical plots, heatmaps, pairplots
  • Plotly for interactive charts (intro)
M-04 Statistics & Math for ML Weeks 3–4
  • Descriptive stats: central tendency, dispersion, skew, kurtosis
  • Distributions: normal, binomial, Poisson, uniform, exponential
  • Bayes' theorem & conditional probability
  • Sampling & Central Limit Theorem
  • Hypothesis testing: z, t, chi-square, ANOVA, p-values
  • Confidence intervals; A/B testing
  • Correlation vs causation; covariance
  • Linear algebra & calculus for ML: gradients, chain rule, gradient descent
M-05 Data Wrangling, EDA & SQL Week 5
  • Cleaning workflows; outlier detection (IQR, z-score)
  • Feature engineering: one-hot / label / target encoding
  • Scaling: standard, min-max, robust; binning
  • Imbalanced data: SMOTE, class weights
  • SQL: joins, window functions, CTEs, aggregations
  • Full EDA case study on a real dataset
M-06 Machine Learning Weeks 6–9

Supervised

  • Linear & polynomial regression (OLS + gradient descent)
  • Regularization: Ridge, Lasso, ElasticNet
  • Logistic regression, KNN, Naive Bayes
  • Decision trees (Gini, entropy, pruning)
  • Bagging & Random Forest
  • Boosting: AdaBoost, Gradient Boosting, XGBoost, LightGBM, CatBoost
  • SVM: margins & kernels

Unsupervised

  • K-Means, hierarchical clustering, DBSCAN, GMM
  • PCA, t-SNE, UMAP
  • Anomaly detection

Evaluation & Tuning

  • Cross-validation (k-fold, stratified)
  • Metrics: precision/recall/F1, ROC-AUC, RMSE, MAE, R²
  • Bias–variance tradeoff; over/underfitting
  • GridSearch, RandomSearch, Optuna
  • sklearn Pipelines & ColumnTransformer
  • Interpretability: feature importance, SHAP, LIME
M-07 Deep Learning with PyTorch Weeks 10–13

Foundations

  • Perceptron → MLP; activations (ReLU, sigmoid, softmax, GELU)
  • Forward & backprop — built from scratch in NumPy
  • Losses (MSE, cross-entropy); optimizers (SGD, Adam, AdamW)
  • Dropout, batch/layer norm, early stopping, LR scheduling
  • PyTorch: tensors, autograd, nn.Module, DataLoader, training loops

Computer Vision

  • CNNs: convolutions, pooling, padding, stride
  • Architectures: LeNet → ResNet → EfficientNet
  • Transfer learning & data augmentation
  • Object detection (YOLO) & segmentation (U-Net) — intro

NLP & Transformers

  • Tokenization; embeddings (Word2Vec, GloVe)
  • RNN, LSTM, GRU; seq2seq + attention
  • Transformers in depth: self-attention, multi-head, positional encoding
  • BERT vs GPT; fine-tuning with Hugging Face
M-08 MLOps Basics Week 14
  • Experiment tracking: MLflow / Weights & Biases
  • Model serialization & versioning
  • Serving with FastAPI; Docker basics
  • Streamlit / Gradio demos; cloud deployment overview
Course Projects
  • End-to-end ML project (e.g., churn prediction) — EDA → model → deployed API
  • CNN image classifier with transfer learning
  • NLP project: fine-tuned Transformer for text classification
